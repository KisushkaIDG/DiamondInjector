Diamond Inject is a utility for Windows DLL injection. The program is written entirely in C and Windows API. It comes in x32 and x64 bits, for injection into processes of different bitness. The program receives the path to the DLL, and then the PID of the process, after which it performs the injection. All instructions for building the project and using it are available in the docs folder, in diamond_inject.txt.
The utility is suitable for injecting DLLs for software development purposes, and it is also suitable for use by security professionals and antivirus developers, as well as for loading cheats or modifications into games and applications. Diamond Inject is one of the few open-source programs of its kind. It is often the case that a virus is embedded in a program under the guise of an injector. Diamond Inject has an open-source license, the GNU General Public License (GPL).
